MRC Loader

This is a proof-of-concept MRC file parser and visualization tool. Click and drag the mouse to view the extracted MRC data from multiple angles. Zoom in and out with your scroll wheel.

Click the 'Browse' button to upload and view your own MRC file.

Density threshold:

The header information is also written to the screen. If you want, you can download the default MRC file which is shown in the tool. Implementation specifics can be found by looking at the source code.

Use the "density threshold" input to specify the minimum density necessary to draw the voxel. Press 'enter' after specifying the number to update the visualization.
